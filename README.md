# My custom Nvchad configuration

### Featuring:

- Ideal setup for Windows with WSL
- Modularised setup of plugins and configurations
- Pair with Wezterm using
  <a href="https://github.com/mgastonportillo/wezterm-config">my Wezterm
  configuration</a> for a smooth experience
- Bash language server & Tree Sitter syntax
- Web Development with React fully covered (using Node)
- Development with Python (code completion yet to be implemented)
- Development with Rust (fully implemented)
- Development with C has a the minimal, default NvChad configuration
- Git fully integrated using Gitsigns and Vim Fugitive

<hr>

### Planned implementations:

- Markdown Preview
- Python code completion
- Auto-installation (by now it's just a back up configuration)

<hr>

### Requirements:

- NVIM v0.9.1
- Default Neovim directory
- NvChad v2.0
- Python 3.10.6 & pip
- Python venv properly configured (nvim/venv directory to target
  $HOME/nvim/venv/bin/activate)
- pip: `pynvim==0.4.3`
- Node.js v20.2.0 & npm (sorry, no yarn support)
- npm: `neovim@4.10.1`
- npm: `typescript@5.1.3`
- Cargo 1.69.0
